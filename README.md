# leetcode-enforcer

A personal macOS forcing-function tool: periodically presents a LeetCode problem in a
full-screen blocker and won't release until you submit an **Accepted** solution. Includes
a Socratic LLM helper (local Ollama/qwen) that gives hints, never full answers.

See [DESIGN.md](DESIGN.md) for the full design.

> Phase 1 = Python prototype to validate the habit loop. Phase 2 = Swift rewrite for real
> enforcement (only if the prototype proves it sticks).

## Status
Design complete; prototype not started.
